VIDEO STREAMING WEBSITE DOCUMENTATION
====================================

1. APP.PY (Main Application File)
--------------------------------
This is the main file that runs the website. It contains:

a) Setup and Configuration:
   - Uses Flask framework
   - Sets up database (SQLite)
   - Configures upload limits (500MB max)
   - Sets allowed video formats (mp4, avi, mov, wmv, flv, mkv)
   - Creates necessary folders (uploads, thumbnails)

b) User System:
   - Registration and login
   - Password security (hashed passwords)
   - Superuser role for admin features
   - Session management

c) Video Management:
   - Upload videos 
   - Delete videos 
   - Watch videos (all logged-in users)
   - Automatic thumbnail generation
   - File size and format validation

d) Admin Features:
   - User management
   - Video management
   - Access control for admin functions

2. TEMPLATES (HTML Files)
------------------------
a) base.html:
   - Main layout template
   - Navigation menu
   - Theme toggle (dark/light mode)
   - Responsive design

b) home.html:
   - Displays video list
   - Shows video thumbnails
   - Upload and delete options for superusers
   - Login/register prompts for guests

c) login.html:
   - User login form
   - Error messages
   - Link to registration

d) register.html:
   - New user registration
   - Username validation
   - Password requirements

e) upload.html:
   - Video upload form
   - Title input
   - File selection
   - Format validation

f) watch.html:
   - Video player
   - Video title
   - Back to home option

g) admin_dashboard.html:
   - User list with delete options
   - Video list with delete options
   - Access control messages

6. HOW TO USE THE WEBSITE
------------------------
a) For Normal Users:
   1. Register an account
   2. Log in
   3. Browse videos
   4. Watch videos
   5. Toggle theme

b) For Superusers:
   1. Log in with superuser account
   2. Upload videos
   3. Delete videos
   4. Manage users
   5. Access admin dashboard
